Atividade Prática: Consultas SQL

Introdução ao Desafio

Objetivo:

Praticar o uso dos comandos `SELECT` e `WHERE` para extrair informações do nosso banco de dados.

Instruções:

  1. Leia a pergunta no slide.
  2. Tente escrever o comando SQL e encontrar a resposta por conta própria.
  3. Avance para o slide seguinte para ver o gabarito e a explicação.

Sumário da Atividade

Nosso Banco de Dados

Estrtutura básica:

  • Pessoa (id, nome, fone)
    Contém Pessoas Físicas e Jurídicas.
  • Area (id, nome)
    Ex: 'Tecnologia da Informação', 'Saúde'.
  • Especialidade (id, nome, area_id)
    Ex: 'Banco de Dados', 'Cardiologia'.
  • PessoaFisica (cpf, pessoa_id)
    Contém os CPFs ligados ao ID da Pessoa.
  • PessoaJuridica (cnpj, pessoa_id)
    Contém os CNPJs ligados ao ID da Pessoa.
  • Projeto (id, nome, data_inicio, data_fim, coordenador_cpf)
    Contém os projetos, datas e quem coordena.
  • Doacao (id, valor, data, projeto_id, pessoajuridica_cnpj)
    Contém doações feitas por PJs para projetos.

Questão 1

Liste os nomes de todas as Áreas de conhecimento cadastradas.

Gabarito: Questão 1

Comando SQL:

SELECT nome FROM Area;

Resposta (Resultado):

(Uma lista de 30 áreas, incluindo 'Tecnologia da Informação', 'Saúde', 'Educação', etc.)

Explicação do Comando:

  • SELECT nome: Informa ao banco que queremos selecionar (buscar) os dados da coluna `nome`.
  • FROM Area: Especifica de qual tabela queremos buscar os dados, neste caso, a tabela `Area`.

Questão 2

Liste o nome e o telefone de todas as Pessoas (físicas e jurídicas) cadastradas.

Gabarito: Questão 2

Comando SQL:

SELECT nome, fone FROM Pessoa;

Resposta (Resultado):

(Uma lista de 60 nomes e telefones, começando com 'Ana Silva | 85911110001' e 'Empresa Alpha TI | 88922220001', etc.)

Explicação do Comando:

  • SELECT nome, fone: Seleciona múltiplas colunas. Pedimos ao banco para buscar tanto a coluna `nome` quanto a coluna `fone`.
  • FROM Pessoa: Especifica que a busca deve ser feita na tabela `Pessoa`.

Questão 3

Qual é o nome da Pessoa (física ou jurídica) que possui o id = 15?

Gabarito: Questão 3

Comando SQL:

SELECT nome FROM Pessoa WHERE id = 15;

Resposta (Resultado):

Olívia Queiroz

Explicação do Comando:

  • SELECT nome: Queremos buscar a coluna `nome`.
  • FROM Pessoa: Da tabela `Pessoa`.
  • WHERE id = 15: Esta é a condição (filtro). Estamos dizendo ao banco: "Não traga todos os nomes, traga apenas aquele ONDE (WHERE) o `id` é igual a 15".

Questão 4

Qual é o telefone da empresa (Pessoa) chamada 'Construtora Delta'?

Gabarito: Questão 4

Comando SQL:

SELECT fone FROM Pessoa WHERE nome = 'Construtora Delta';

Resposta (Resultado):

88922220004

Explicação do Comando:

  • SELECT fone: Queremos buscar a coluna `fone`.
  • FROM Pessoa: Da tabela `Pessoa`.
  • WHERE nome = 'Construtora Delta': Filtramos pela coluna `nome`. Note que textos (strings) em SQL devem estar entre aspas simples (' ').

Questão 5

Qual é o pessoa_id (ID da tabela Pessoa) associado à Pessoa Física com CPF '00000000001'?

Gabarito: Questão 5

Comando SQL:

SELECT pessoa_id FROM PessoaFisica WHERE cpf = '00000000001';

Resposta (Resultado):

1

Explicação do Comando:

  • SELECT pessoa_id: Seleciona a coluna `pessoa_id`.
  • FROM PessoaFisica: Busca na tabela `PessoaFisica`.
  • WHERE cpf = '00000000001': Filtra para encontrar a linha exata onde o `cpf` corresponde ao valor procurado.

Questão 6

Qual é o nome da Especialidade que possui o id = 10?

Gabarito: Questão 6

Comando SQL:

SELECT nome FROM Especialidade WHERE id = 10;

Resposta (Resultado):

Botânica

Explicação do Comando:

  • SELECT nome: Seleciona a coluna `nome`.
  • FROM Especialidade: Busca na tabela `Especialidade`.
  • WHERE id = 10: Filtra pelo `id` número 10 (números não precisam de aspas).

Questão 7

Liste o nome de todos os projetos que começaram (ou vão começar) exatamente no dia '2024-01-01'.

Gabarito: Questão 7

Comando SQL:

SELECT nome FROM Projeto WHERE data_inicio = '2024-01-01';

Resposta (Resultado):

Sistema de Biblioteca

Explicação do Comando:

  • SELECT nome: Seleciona a coluna `nome` (nome do projeto).
  • FROM Projeto: Busca na tabela `Projeto`.
  • WHERE data_inicio = '2024-01-01': Filtra pela coluna `data_inicio`. Datas em SQL geralmente são tratadas como strings (texto) no formato 'AAAA-MM-DD'.

Questão 8

Liste apenas os valores de todas as doações que foram maiores que R$ 20.000.

Gabarito: Questão 8

Comando SQL:

SELECT valor FROM Doacao WHERE valor > 20000;

Resposta (Resultado):

50000.00
100000.00
30000.00
45000.00
22000.00

Explicação do Comando:

  • SELECT valor: Seleciona a coluna `valor`.
  • FROM Doacao: Busca na tabela `Doacao`.
  • WHERE valor > 20000: Aqui usamos um operador de comparação (>). Queremos as linhas ONDE (WHERE) o `valor` é maior que 20000.

Questão 9

Qual foi o projeto_id que recebeu uma doação da empresa (Pessoa Jurídica) com CNPJ '00000000000019'?

Gabarito: Questão 9

Comando SQL:

SELECT projeto_id FROM Doacao WHERE pessoajuridica_cnpj = '00000000000019';

Resposta (Resultado):

19

Explicação do Comando:

  • SELECT projeto_id: Seleciona a coluna `projeto_id`.
  • FROM Doacao: Busca na tabela `Doacao`.
  • WHERE pessoajuridica_cnpj = '00000000000019': Filtra a tabela `Doacao` pela coluna `pessoajuridica_cnpj` para encontrar o CNPJ específico.

SELECT nome FROM Projeto WHERE coordenador_cpf = '00000000010';

Resposta (Resultado):

Otimização Cálculo Numérico

Explicação do Comando:

Fim da Atividade

Parabéns por completar o exercício!

O que aprendemos:

  • Buscar colunas específicas com SELECT.
  • Escolher a tabela correta com FROM.
  • Filtrar por números com WHERE id = 10.
  • Filtrar por texto com WHERE nome = '...'.
  • Usar operadores de comparação como WHERE valor > 20000.
